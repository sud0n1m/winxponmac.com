Windows XP on the Intel Mac


Bootcamp



Soon after the release of XOM, Apple made several steps to correct their original error of not making it easy to install windows:


  2. Apple released a firmware update for Intel Macs that added a bios emulation module, thus enabling booting operating systems that used BIOS rather than the newer EFI standard that Apple had adopted.

  4. Apple released "Bootcamp", software to help users non-destructively repartition their hard drive

  6. Bootcamp burned a Drivers CD that fixed earlier problems XOM had with hardware recognition

  8. Bootcamp's boot manager allowed users to select which OS to boot (a feature matched by XOM)


These releases from Apple eliminated the need for a solution like XOM and put the responsibility in the hands of those who should have it!
